SpreeAffiliate
==============


Installation
============

1. Add to Gemfile:

        gem "spree_store_credits"
        gem "spree_email_to_friend"
        gem "spree_affiliate"

1. Run `bundle install`
1. Run install rake task for all extensions:

        rake spree_store_credits:install
        rake spree_affiliate:install
  
1. Run `rake db:migrate`
